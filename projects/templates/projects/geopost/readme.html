Introduction

GeoPost is a web map blog. It's pretty simple, you just click on a point on the map and the text and photo will appear for you to view.

Under the Hood

The Stack:

  • OpenLayers3, the Javascript mapping library

  • GeoServer servlet in an Apache Tomcat container

  • PostGIS geospatial database extension for PostgreSQL

Essentially just the standard web GIS stack, with media storage accomplished using Google Cloud Storage. The geometry and other attributes of the entries are stored in a PostGIS relation, and are served by GeoServer over WFS to the browser, where they are displayed on the map by OpenLayers.
