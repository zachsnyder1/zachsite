Introduction

Zach-Site.com is where I document all of my active projects, and where I host my web applications. Below you will find a brief description of the stack with which the site is built. Code for all projects is viewable on my GitHub--see the black navbar above for links.

Server Side

The Stack:

  • Django, the Python web framework

  • PostgreSQL database

  • Apache Httpd web server

  • Ubuntu Linux

  • Google Cloud Platform IAAS

Django is the key framework of the project, and the site is deployed manually on Google Compute Engine VMs running Ubuntu Linux. Unit testing is performed with Django's extension of the Python standard library unittest module.

Client Side

Built With:

  • Twitter Bootstrap, the mobile-first front end framework

  • Sass, the CSS extension language

  • Grunt, the Node.js task runner. Used mainly for compiling SCSS files and running QUnit unit tests

I have enjoyed getting my hands dirty with some of the CSS and Javascript. I've used Bootstrap as the real workhorse, but I've tried to customize the look and feel in ways that I think create value. I don't know, you tell me.... UI level testing is automated with Selenium Webdriver.
